# cn-portal-hub

## 项目简介

`cn-portal-hub` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，而是作为一个通用容器，集中存放各类静态页面资源。每个页面均为独立、自包含的 HTML 文件，可直接通过浏览器打开或部署至静态托管服务。

## 目录说明

```
cn-portal-hub/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面引用的公共资源（可选，如样式、脚本、图片等）
├── index.html      # 仓库首页，提供页面导航或简单说明
├── README.md       # 本说明文件
└── .gitignore      # Git 忽略规则
```

- **pages/**：主要归档目录，每个 HTML 文件对应一个独立页面，命名清晰以便识别。
- **assets/**：若页面需要共用资源（如 CSS、JS、图片），可统一放置于此，避免重复。
- **index.html**：可选入口页面，可列出所有归档页面的链接，方便浏览。

## 页面归档说明

- 所有页面均为纯前端 HTML，不依赖后端服务。
- 每个页面应保持独立，避免跨文件依赖（公共资源除外）。
- 页面内容应遵守相关法律法规，不得包含恶意代码、侵权内容或违反平台政策的信息。
- 提交新页面时，请确保文件命名具有辨识度，并在 `index.html` 中更新链接（如需要）。

## 维护说明

- 本仓库由维护者定期更新，欢迎通过 Issue 或 Pull Request 提交新的归档页面。
- 提交前请检查页面是否能在主流浏览器上正常显示。
- 若页面引用了外部资源（如 CDN 库），请确保其来源可靠且长久有效。
- 维护者保留对不符合归档要求的内容进行修改或删除的权利。
